
[1] file 관리하는 명령어
1. 학습 순서
    1) file 생성 및 데이터 저장
    2) file 이동 및 삭제


2. file 생성 및 내용 보기
    방법 1) vi 텍스트 편집기 사용
        $ vi text.txt
    방법 2) 단순 확인
        $ cat text.txt
    방법 3) 소량의 데이터를 하나의 명령어로 파일 생성 가능
        $ echo "데이터" > test.txt
    방법 4) 한 line씩 이동하면서 확인 가능한 명령어
        $ less test2.txt
        방향키 또는 enter로 line by line 확인
        파일 보기 중지 시에는 q
    방법 5)
        $ head -n 숫자 파일명
        $ tail -n 숫자 파일명
    방법 6) 엔터로만 다음 내용 확인 가능, 문서의 몇%가 보여지는지 확인. 종료 시 ctrl+z
        $ more test2.txt
    방법 7) 빈 파일 생성, 다중 파일 생성도 가능
        $ touch 파일명.확장자 파일명.확장자


3. 디렉토리 내용 확인
    1) 단순 확인 - 파일인지 디렉토리인지 불명확
        $ ls
    
    2) 사용자가 생성한 내용만 확인
        $ ls -l

    3) linux 자체의 설정 파일까지 확인
        $ ls -al


4. file 이동 및 삭제
    현 디렉토리에 있는 파일 이동 명령어
    $ mv 파일명 디렉토리명/

    현 경로 상에서 하위 디렉토리 내용 확인
    $ ls 디렉토리명/

    파일 삭제
    $ rm 파일명

    복사/붙여넣기
    $ cp test2.txt poem.txt
    $ cp 원본파일 (디렉토리명/)복제파일명


4. file에 데이터 입력에 따른 명령어 및 키워드ㅡ
    사용 기본 프로그램이 vi 편집기
    vi 편집기 필수 키워드
        파일 오픈해서 새로운 데이터 추가 작업 시에 필수인 표현법
        i = insert의 약자
        a = append의 약자
        esc -> :wq! = 수정된 내용 저장 및 종료
        esc -> :q! = 수정된 사항 무시하고 저장 없이 종료
        dd = 커서가 있는 line의 데이터 삭제 의미

        작업 오류 생겼을 경우)
            esc 여러 번 누르기 & ctrl+c :q!
        입력 시도
            잘 안 될 경우 a or i 입력

* 특정 단어 포함 문장 검색 명령어
    $ grep 윤동주 encore/dongju.txt
