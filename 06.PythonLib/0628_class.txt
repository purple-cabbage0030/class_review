5. 웹 크롤링 + 공공데이터 & open API
    웹 상의 자료 수집(크롤링)/선별(데이터 전처리)해서 데이터 베이스에 저장(sq 기반 crud)

    서비스 제공자 관점에서 데이터 취합해서 보관
    - 크롤링: html / css / java script 볼 수 있는 범위 내에서 데이터 취합
    - 선별(데이터 전처리): Pandas & 시각화 라이브러리
    - db에 저장: sql + python

    취합된 데이터를 end user에게 서비스
    - db -> crud -> 브라우저(url(rest API) + 요청 + 응답 + JSON + 비동기)
